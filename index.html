How to play

  • Press start/pause to start or pause the game :)
  • press ⬇ (or arrow down key on keyboard) to go down
  • press ⬆ (or arrow up key on keyboard) to change the orientation of the tetromino
  • press ⬅ (or arrow left key on keyboard) to move the tetromino to the left
  • press ➡ (or arrow right key on keyboard) to move the tetromino to the right
